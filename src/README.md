- **Purpose:** reusable Python modules for data loading, feature engineering, and utilities.

- **Main modules:**
  - `data_loader.py` — loads and cleans datasets.
  - `features.py` — feature engineering: time features, scaling, encoding, resampling.
  - `geolocation.py` — IP-to-country mapping utilities.

- **How to use:** import modules in notebooks or scripts, e.g. `from src.data_loader import load_fraud_data`.
